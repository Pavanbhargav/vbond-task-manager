# Project Overview
This project uses a separated frontend and backend architecture.

## Frontend (`/client`)
- Framework: Next.js with React
- UI Library: Shadcn UI
- Rule: Always run `npm` (or `pnpm`) commands inside the `/client` directory. Do not write frontend code or install packages in the root directory.

## Backend (`/server`)
- Framework: Django
- Rule: Always run `python manage.py` or `pip` commands inside the `/server` directory.
